# Análise da Agricultura Familiar no Ceará

**Integrante:** [Seu nome]  
**Tema:** Impacto da assistência técnica e cooperativas na produção e renda de assentados.

## Objetivo
Dashboard para identificar quais fatores (assistência técnica, cooperativa, localização) aumentam a produtividade e renda de famílias assentadas no Ceará.

## Fontes de dados
1. Dataset próprio simulando 2000 famílias (baseado em parâmetros reais do semiárido).
2. Preços CEASA – arquivo público: https://files.ceasa-ce.com.br/unsima/boletim_diario/maracanau/Tabela_de_Preços_Max_Min_Méd.csv

## Instruções para abrir
- Baixe o arquivo `dashboard/projeto_final.pbix` e abra com Power BI Desktop.
- Os dados já estão embutidos; não é necessário baixar arquivos externos.

## Principais insights
- Famílias com assistência técnica produzem 34% mais.
- Cooperativas aumentam a renda em média 22%.
- Milho e mandioca representam 68% da produção total.
